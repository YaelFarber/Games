# JavaScript DOM Games Project

### Project Description
This project is a browser-based JavaScript application consisting of two interactive games, developed using pure JavaScript as part of the JavaScript + DOM (Part B) curriculum.

The project demonstrates practical use of DOM manipulation, event handling, animations, and browser storage, without using external libraries or frameworks.

---

### Included Games
- **Strawberry Dodger** – a real-time reflex game where the player collects strawberries and avoids falling broccolies, featuring dynamic movement, collision detection, scoring, and increasing difficulty.
- **Rush Hour** – 10 levels rush hour game focused on movement control, and continuous DOM updates during gameplay.

---

### Academic Context
- **Course:** Full-Stack Web Development  
- **Institution:** Lev Academic Center (JCT)  

---

### Implemented Features
- Dynamic DOM creation and manipulation  
- Event-driven game logic (keyboard and UI events)  
- JavaScript-based animations and timing  
- Game state and score management  
- Persistent data using `localStorage`  
- Clear separation between HTML, CSS, and JavaScript  

---

### Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- DOM API  
- Web Storage API  

---

### How to Run
1. Open `Games/index.html` in a modern web browser  
2. No installation or server setup is required
